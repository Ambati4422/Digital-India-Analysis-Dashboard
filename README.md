Digital India Analysis Dashboard
📌 Project Overview
The Digital India Analysis Dashboard is an interactive Power BI project designed to analyze digital transformation across different states in India.
This dashboard provides valuable insights into:
Customer distribution
UPI usage trends
Internet usage by state
Aadhaar linked percentage
Digital literacy by age group
